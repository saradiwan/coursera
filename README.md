# Task API Project

## Description
A simple REST API built using Node.js and Express for managing tasks.

## Features
- Add task (POST /tasks)
- View all tasks (GET /tasks)
- Delete task (DELETE /tasks/:id)

## How to run

1. Install dependencies:
npm install

2. Start server:
npm start

3. Open:
http://localhost:3000

## Example POST body:
{
  "task": "Learn API development"
}
